# Harmonising European Real-World Data for Regulatory Evidence Generation

This repository contains the code used for the article:

**Harmonising European Real-World Data for Regulatory Evidence Generation: OMOP-Based Analyses of Breast Cancer and Amyotrophic Lateral Sclerosis**

## Repository structure

The repository is organized into the following folders:

- `ETL/`  
  Contains database functions used as part of the ETL scripts, as well as utility functions.

- `Analysis/`  
  Contains example analysis scripts for the amyotrophic lateral sclerosis (ALS) and breast cancer (BC) analyses.

## Data availability

References to the original source data have been removed from the scripts in accordance with data governance and access policies. Therefore, the repository does not include source data or any data-specific access information.

## Notes

The code is provided to document the analytical workflow used in the article and to support transparency and reproducibility where possible, within the constraints of the applicable data policies.
